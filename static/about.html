ChainedLink is like LinkedIn, except your profile is stored on a blockchain, and you maintain your own connections to others.

Actually, some data is stored directly on a Blockchain, and other data (like longer bio text, avatar images, etc) are stored in public Github repositories and simply linked to (via SHA1 git commit hashes) from records in the Blockchain.
